# tf-jupyter-docker

Containerized Jupyter notebook with TensorFlow.

To build:

`./build.sh`

To run:

`./start.sh`

Default Python3 Packages (latest unless specified):

```
    pip \
    setuptools \
    matplotlib \
    hdf5storage \
    h5py \
    pandas \
    seaborn \
    scikit-learn \
    jupyter
``
